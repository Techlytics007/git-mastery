# Sandbox

Free-play area for experimentation. No rules here.

## How to Use

Create subdirectories for different experiments:

```bash
cd sandbox
mkdir my-experiment && cd my-experiment
git init
# ... experiment freely ...
```

Or use the `sandbox/` directory itself — it's gitignored for `scratch/` subdirectory:
```bash
mkdir sandbox/scratch   # fully ignored by git
```
